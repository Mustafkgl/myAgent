## 📦 Installation and Power Modes

### 🐳 Docker (Maximum Autonomy & Security)
*In this mode, the agent uses commands like `g++`, `sed`, `mkdir` with full authority but in isolation.*

```bash
docker compose build
./run.sh
```

### 🐍 Local (Quick Start)
```bash
pip install -e .
python -m myagent
```
